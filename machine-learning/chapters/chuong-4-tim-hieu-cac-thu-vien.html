CHƯƠNG 4. TÌM HIỂU CÁC THƯ VIỆN PYTHON CƠ BẢN

Khám phá bộ 4 thư viện trụ cột bắt buộc phải nắm vững khi bắt đầu học máy: NumPy, Pandas, Matplotlib/Seaborn và Scikit-Learn.

Ngôn ngữ lập trình Python cung cấp một hệ sinh thái các thư viện cực kỳ phong phú phục vụ cho việc xử lý dữ liệu, phân tích thống kê và trực quan hóa đồ thị. Thư viện (Library) là tập hợp các hàm và đối tượng đã được các chuyên gia viết sẵn, giúp chúng ta sử dụng lại một cách tối ưu thay vì phải lập trình lại mọi thứ từ đầu.

Khi mới bắt đầu bước chân vào lĩnh vực Machine Learning, có 4 thư viện bắt buộc phải biết:

NumPy

Xử lý số liệu, tính toán ma trận đa chiều.

Pandas

Thao tác dữ liệu dạng bảng (Excel/CSV).

Matplotlib/Seaborn

Trực quan hóa, vẽ đồ thị & biểu đồ.

Scikit-Learn

Huấn luyện thuật toán Machine Learning.

4.1 NumPy - Nền tảng xử lý số liệu

4.1.1 Định nghĩa

NumPy (Numerical Python) giúp thực hiện các phép toán số học hiệu năng cao trên dữ liệu mảng đa chiều.

Trong Machine Learning, dữ liệu thô đầu vào (như ảnh chụp, tệp âm thanh hay văn bản) không ở dạng số trực tiếp. NumPy đóng vai trò chuyển đổi và biểu diễn toàn bộ dữ liệu này dưới dạng các ma trận số để máy tính có thể thực hiện huấn luyện.

4.1.2 Ví dụ minh họa với NumPy

Ví dụ 1: Tạo mảng số 1 chiều

import numpy as np

# Tạo mảng 1 chiều
arr = np.array([1, 2, 3, 4, 5])
print(arr)
# Kết quả: [1 2 3 4 5]

Ví dụ 2: Nhân hai ma trận hai chiều

import numpy as np

A = np.array([[1, 2],
              [3, 4]])

B = np.array([[5, 6],
              [7, 8]])

C = A @ B   # Toán tử @ thực hiện nhân ma trận
print(C)
# Kết quả:
# [[19 22]
#  [43 50]]

4.2 Pandas - Làm việc với dữ liệu dạng bảng

4.2.1 Pandas và DataFrame là gì?

Trong thực tế, dữ liệu Machine Learning thường tồn tại ở dạng bảng (giống như tệp Excel, CSV, hoặc bảng SQL). Pandas là thư viện dùng để đọc, thao tác, làm sạch và phân tích dữ liệu dạng bảng này.

Cấu trúc dữ liệu cốt lõi nhất của Pandas là DataFrame – một bảng dữ liệu gồm:

  • Hàng (Rows): Mỗi hàng đại diện cho một bản ghi dữ liệu (ví dụ: thông tin của một người).
  • Cột (Columns): Mỗi cột đại diện cho một đặc trưng (Feature) của dữ liệu (ví dụ: tuổi, mức lương).

4.2.2 Ví dụ minh họa với Pandas

Ví dụ 1: Tạo một DataFrame

import pandas as pd

data = {
    "name": ["An", "Bình", "Chi", "Dũng"],
    "age": [22, 35, 28, 42],
    "salary": [500, 1200, 800, 1500]
}

df = pd.DataFrame(data)
print(df)
# Kết quả DataFrame được tạo ra dưới dạng bảng chỉ mục

Ví dụ 2: Thống kê nhanh mô tả trên DataFrame

# Thống kê nhanh các chỉ số toán học
print(df.describe())

4.3 Matplotlib / Seaborn - Trực quan hóa dữ liệu

4.3.1 Matplotlib và Seaborn là gì?

Matplotlib và Seaborn giúp trực quan hóa kết quả phân tích số liệu dưới dạng biểu đồ cột, biểu đồ phân phối, đường xu hướng để con người dễ dàng quan sát và rút ra kết luận.

4.3.2 Ví dụ trực quan hóa

Ví dụ 1: Vẽ biểu đồ so sánh lương (Matplotlib)

import matplotlib.pyplot as plt

names = ["An", "Bình", "Chi", "Dũng"]
salary = [500, 1200, 800, 1500]

plt.bar(names, salary)
plt.xlabel("Tên")
plt.ylabel("Lương")
plt.title("So sánh lương theo từng người")
plt.show()

4.4 Scikit-learn – Thư viện Machine Learning cơ bản

4.4.1 Scikit-learn là gì? Dùng khi nào?

Scikit-Learn (sklearn) là thư viện học máy phổ biến nhất trong Python, đóng gói sẵn hầu hết các thuật toán ML cổ điển cũng như các công cụ xử lý dữ liệu đầu vào.

Thư viện này được áp dụng cho 3 bài toán cổ điển chính:

  • Regression (Hồi quy): Dự đoán đại lượng số thực.
  • Classification (Phân loại): Dự đoán nhãn rời rạc.
  • Clustering (Phân cụm): Gom cụm các đối tượng tương đồng.

4.4.2 Ví dụ minh họa với Scikit-learn

Ví dụ 1: Bài toán hồi quy tuyến tính (Dự đoán lương theo tuổi)

from sklearn.linear_model import LinearRegression
import numpy as np

# Chuẩn bị dữ liệu đầu vào (X: tuổi, y: lương)
X = np.array([[22], [35], [28], [42]])  # Tuổi (đặc trưng dạng mảng 2D)
y = np.array([500, 1200, 800, 1500])    # Lương (nhãn mục tiêu)

# 1. Khởi tạo mô hình hồi quy tuyến tính
model = LinearRegression()

# 2. Huấn luyện mô hình (Tìm hàm số tối ưu đi qua các điểm dữ liệu)
model.fit(X, y)

# 3. Dự đoán mức lương cho người 30 tuổi
pred = model.predict([[30]])
print(pred)
# Kết quả dự đoán lương: [922.9]

Ví dụ 2: Bài toán phân loại nhị phân (Đỗ / Trượt dựa trên điểm số)

from sklearn.linear_model import LogisticRegression

# Dữ liệu huấn luyện (Điểm số)
X = [[5], [6], [7], [8], [9]]
y = [0, 0, 0, 1, 1]  # 0: rớt, 1: đậu

# 1. Khởi tạo mô hình hồi quy Logistic
model = LogisticRegression()

# 2. Huấn luyện mô hình
model.fit(X, y)

# 3. Dự đoán xác suất cho học sinh được 7.5 điểm
print(model.predict([[7.5]]))
# Kết quả đầu ra: [0] (Hệ thống dự đoán trượt vì điểm 7.5 nằm sát ngưỡng rớt)
