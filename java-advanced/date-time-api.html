17.1 LocalDate (Lớp biểu diễn Ngày)

Khái niệm và mục đích sử dụng

LocalDate là lớp biểu diễn một ngày cụ thể (năm-tháng-ngày) trong hệ thống lịch ISO-8601. Lớp này hoàn toàn **không chứa thông tin về Giờ, Phút, Giây và Múi Giờ**.

Ứng dụng thực tế: Thích hợp lưu trữ các thông tin ngày mang tính độc lập như: Ngày sinh nhật, ngày thành lập công ty, ngày hết hạn bảo hành của sản phẩm.

Cơ chế hoạt động bên trong (Tính bất biến - Immutability)

Tương tự như lớp String, các lớp trong bộ Date Time API mới của Java 8 đều là **Immutable Objects (Đối tượng bất biến)** và hoàn toàn **Thread-safe**.

Mỗi khi bạn thực hiện các phép cộng trừ ngày, JVM không bao giờ sửa đổi giá trị ô nhớ của đối tượng hiện tại, mà luôn tự sinh và trả về một đối tượng mới hoàn toàn:

LocalDate today = LocalDate.now(); (Lưu tại ô nhớ 0x101) today.plusDays(5); (Sinh ra đối tượng mới tại ô 0x202, today tại 0x101 giữ nguyên giá trị)
Cú pháp và ví dụ mã nguồn
import java.time.LocalDate;

LocalDate today = LocalDate.now(); // Lấy ngày hiện tại hệ thống
LocalDate birthday = LocalDate.of(2004, 10, 15); // Khởi tạo ngày cụ thể

// Cách sửa lỗi phổ biến (Phải gán lại giá trị):
today = today.plusDays(7); // today nhận giá trị mới cộng thêm 7 ngày

17.2 LocalTime (Lớp biểu diễn Thời gian)

Khái niệm và mục đích sử dụng

LocalTime đại diện cho thời gian trong ngày (giờ, phút, giây, nanosecond) và **không chứa thông tin ngày tháng năm hay múi giờ**.

Thường dùng biểu diễn: Giờ mở cửa/đóng cửa cửa hàng, giờ bắt đầu các ca làm việc cố định.

Ví dụ mã nguồn đơn giản
import java.time.LocalTime;

LocalTime now = LocalTime.now();
LocalTime startTime = LocalTime.of(8, 30); // 08:30:00

LocalTime afterTwoHours = now.plusHours(2);
System.out.println(afterTwoHours);

17.3 LocalDateTime (Ngày & Giờ kết hợp)

Khái niệm và mục đích sử dụng

LocalDateTime là sự kết hợp đồng thời giữa LocalDate và LocalTime, hiển thị đầy đủ thông tin ngày, tháng, năm cùng với giờ, phút, giây nhưng **không có múi giờ**.

Ứng dụng thực tế: Được sử dụng làm kiểu dữ liệu chuẩn mực để ghi nhận thời điểm xảy ra sự kiện trong hệ thống như: createdAt (thời điểm tạo đơn hàng), updatedAt (thời điểm cập nhật thông tin).

Ví dụ mã nguồn đơn giản
import java.time.LocalDateTime;

LocalDateTime now = LocalDateTime.now();
LocalDateTime orderTime = LocalDateTime.of(2026, 7, 12, 20, 30);

17.4 & 17.5 Đo lường khoảng cách thời gian: Duration vs Period

Để tính toán khoảng thời gian chênh lệch giữa hai thời điểm, Java cung cấp hai lớp bổ trợ tùy thuộc vào đơn vị đo lường:

1. Duration (Khoảng thời gian ngắn tính bằng Giờ/Phút/Giây/Mili)

Làm việc với các lớp chứa thông tin thời gian như LocalTime hoặc LocalDateTime. Thích hợp đo hiệu năng chạy hàm:

LocalDateTime start = LocalDateTime.now();
// Thực thi nghiệp vụ...
LocalDateTime end = LocalDateTime.now();

Duration duration = Duration.between(start, end);
System.out.println("Thời gian chạy (mili giây): " + duration.toMillis());
2. Period (Khoảng thời gian dài tính bằng Năm/Tháng/Ngày)

Làm việc với lớp LocalDate để đo lường khoảng cách ngày tháng. Thích hợp tính tuổi:

LocalDate birthday = LocalDate.of(2004, 10, 15);
LocalDate today = LocalDate.now();

Period age = Period.between(birthday, today);
System.out.println("Bạn đã: " + age.getYears() + " tuổi, " + age.getMonths() + " tháng.");

Parse & Format Ngày tháng (Biến đổi String <=> Date)

Java 8 cung cấp lớp DateTimeFormatter vô cùng mạnh mẽ để phân tích chuỗi và xuất bản ngày tháng theo định dạng hiển thị tùy biến:

1. Parse (Chuyển đổi từ chuỗi String sang LocalDate)
// Sử dụng format mặc định ISO-8601 (yyyy-MM-dd)
LocalDate date = LocalDate.parse("2026-07-12");
2. Format (Chuyển đổi từ LocalDate/LocalDateTime sang String để hiển thị)
import java.time.format.DateTimeFormatter;

DateTimeFormatter formatter = DateTimeFormatter.ofPattern("dd/MM/yyyy HH:mm:ss");
LocalDateTime now = LocalDateTime.now();

String formattedDate = now.format(formatter);
System.out.println("Hiển thị thân thiện: " + formattedDate); // Ví dụ: 12/07/2026 20:30:15
